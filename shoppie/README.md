### Run Locally

To run **Shoppie** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/kilogrammie/shoppie.git
```

Windows:

```bash
git clone https://github.com/kilogrammie/shoppie.git
```
